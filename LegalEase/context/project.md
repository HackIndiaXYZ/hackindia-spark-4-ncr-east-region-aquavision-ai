Project: Legal-Ease

Description:
AI-powered contract analyzer for Indian users.

Core Features:
- Upload PDF
- Detect risky clauses
- Assign severity (Low/Medium/High)
- Calculate risk score (0–100)
- Generate consequences (real-world impact)
- Translate into Hindi, Tamil, Telugu
- Export report

UI:
- Split layout (PDF left, analysis right)
- Risk dashboard with cards
- Accordion for clauses

Tech Stack:
- Next.js (App Router)
- Tailwind CSS + shadcn/ui
- Gemini API (3.1 Pro)

Constraints:
- 36-hour hackathon
- prioritize speed over perfection
- avoid overengineering

Goal:
Build a clean, working demo with strong UI and real-world impact
